## Getting Started
  
### Installation
  
1. Clone this repos (JobHunt.Id)
```sh
git clone https://github.com/ade-ryzki/JobHunId.git
```
2. Go To Folder Repo
```sh
code .
```
3. Install Module
```sh
npm install
```
  
4. Start website
```sh
npm Run Dev
```
